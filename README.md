# blood_donation_simulator

O presente projeto tem como objetivo o desenvolvimento de um aplicativo móvel voltado para a conscientização e incentivo à doação de sangue. A aplicação, desenvolvida em Dart com o framework Flutter, apresenta duas funcionalidades principais: um simulador de aptidão para doação, que orienta o usuário quanto aos critérios básicos exigidos, e um localizador de hemocentros próximos, com base em informações fixas ou integradas a links externos. O projeto contribui para causas sociais relevantes ao facilitar o acesso à informação e aproximar potenciais doadores das instituições de coleta, demonstrando o potencial da tecnologia como instrumento de conscientização e impacto positivo na sociedade.
